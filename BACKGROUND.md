# 自定义背景图片

## 使用方法

1. 准备一张你喜欢的图片（建议尺寸：1920x1080 或更大）
2. 将图片命名为 `bg.jpg`
3. 将图片放到 `public/` 目录下

## 说明

- 支持任何图片格式（.jpg, .png, .webp 等）
- 图片会自动覆盖整个页面背景
- 默认渐变背景会被替换
- 亮色和暗色模式都会使用同一张背景图片

## 示例

```
public/
├── bg.jpg          # 你的自定义背景图片
├── custom.css      # 样式文件
└── ...
```

## 恢复默认渐变背景

如果不想要自定义背景图片，只需：
- 删除 `public/bg.jpg` 文件
- 或者重命名为其他名字（如 `bg.jpg.bak`）

刷新页面后即可恢复默认的渐变背景效果。
