## How to Write

Create a Markdown file in `posts/`:

```
posts/
  2025-05-22-my-post.md
```

Add frontmatter:

```yaml
---
title: Post Title
date: 2025-05-22
tags: [tag1, tag2]
---
```

Use `<!-- more -->` to mark the excerpt cutoff.
